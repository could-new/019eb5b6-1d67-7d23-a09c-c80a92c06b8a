# 待办事项 (Todo List)

一个使用 Flutter 构建的简单而优雅的待办事项管理应用程序。支持在 iOS、Android、Web 以及桌面端运行。

## 功能特性
- **添加任务**：使用悬浮按钮快速添加新的待办事项。
- **标记完成**：通过复选框勾选已完成的任务，完成后会有删除线效果。
- **删除任务**：支持一键移除不需要的待办事项。
- **响应式设计**：在大屏幕（Web、平板、桌面）设备上居中显示并限制最大宽度，确保最佳阅读和操作体验。

## 技术栈
- Flutter (Dart)
- Material 3 Design
- 纯本地状态管理 (StatefulWidget)

## 运行与编译
确保你已经安装了 [Flutter SDK](https://flutter.dev/)。

在终端中执行以下命令运行应用：
```bash
flutter run
```
